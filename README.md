# bisheng for WPS

> **Not an official bisheng product.** This is a community-developed WPS Office plugin.
>
> **非 bisheng 官方出品**，这是一个社区开发的 WPS Office 插件。

bisheng for WPS is a WPS Office client plugin built on the [bisheng](https://github.com/dataelement/bisheng) open-source LLM DevOps platform. It embeds into the WPS sidebar, giving users direct access to all of bisheng's AI capabilities while editing documents — no need to switch to a browser.

bisheng for WPS 是基于 [bisheng](https://github.com/dataelement/bisheng) 开源 LLM DevOps 平台的 WPS Office 客户端插件。它以侧边栏形式嵌入 WPS，让用户在编辑文档时直接使用 bisheng 的全部 AI 能力，无需切换到浏览器。
